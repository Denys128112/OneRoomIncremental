# Top-down gorgon sprites

## Sprite sheets

- `gorgon-1-topdown.png`: basic forest gorgon
- `gorgon-2-topdown.png`: stronger frost gorgon
- `gorgon-3-topdown.png`: elite blood gorgon

All sheets use the same layout:

- Sheet size: `384x256`
- Grid: `12 columns x 4 rows`
- Frame size: `32x64`
- Transparent background

Rows:

1. Down
2. Left
3. Right
4. Up

Columns:

1. Idle 1
2. Idle 2
3. Move 1
4. Move 2
5. Move 3
6. Move 4
7. Attack 1
8. Attack 2
9. Attack 3
10. Hurt
11. Death 1
12. Death 2

`gorgon-1-source.png` is the larger editable generation source.
Use nearest-neighbor scaling when resizing a sheet.
