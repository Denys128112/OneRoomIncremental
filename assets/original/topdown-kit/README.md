# Оригінальні top-down персонажі

Файли скопійовані без змін із головного Craftpix-архіву.

- `heroes/1-3`: три початкові герої;
- `enemies/1-4`: чотири початкові вороги.

У кожній папці містяться оригінальні смуги `Idle`, `Walk`, `Attack`,
`Hurt` і `Death` для напрямків:

- `D`: вниз;
- `S`: убік;
- `U`: вгору.

Для руху праворуч бічні кадри `S` потрібно віддзеркалювати.
